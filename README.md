# RobustEngine
Now Served with a side of hat

A Fully Open Source C# Rendering Engine.

Features:
- Fully Cross platform (developed and tested on linux)
- Supports 2D, 2.5D and 3D graphics.
- Fully Customizable

Upcoming Features:
- Advanced Debugging
- STL/Obj file loading
- Modeling support
- Lighting system
- Advanced Shaders
- Particle Systems
- Physics Systems
